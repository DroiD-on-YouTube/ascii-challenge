# ASCII Challenge

## Goal

Create an [asciinema](https://asciinema.org/)-like version of the iconic Pixar intro.

## Details

The result can be a video or an actual asciinema file created using any programming/scripting language/libraries EXCEPT any that actually do all the work.

For example, [Star Wars Episode IV](https://asciinema.org/a/8).

We have ~2 hours for this challenge. Place your code in the directory with your name. Use the provided `pixar.mp4` as your source.

## Judging criteria

- speed to get a solution
- creativity
- cleverness
- result quality
- extras
